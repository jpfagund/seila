Conceito WEB

Web é uma palavra inglesa que significa teia ou rede. O significado de web ganhou outro sentido com o aparecimento da internet.

O que é WWW?

A web passou a designar a rede que conecta computadores por todo mundo, a World Wide Web (WWW). WWW é o que representa o domínio WEB.

O que é W3C

W3C é quem administra os domínios da WEB
